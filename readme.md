1. Created the backend, and the front end folders
2. Open terminal and navigate to backend
    cd backend
3. Create virtual environment
    python -m venv .venv
    .venv\Scripts\activate
    (.venv) C:\path\backend>   --this means that the environment is active
4. Install flask framework
    pip install flask psycopg2-binary flask-cors
    pip freeze > requirements.txt
5. Create the database and tables.
